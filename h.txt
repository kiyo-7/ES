🇪🇸 Salutations informelles 👋
(amis, famille, personnes du même âge)

• Hola → Salut / Bonjour
• ¡Hola, qué tal! → Salut, ça va ?
• ¿Qué tal? → Ça va ?
• ¿Cómo estás? → Comment vas-tu ?
• ¿Cómo te va? → Comment ça va ?
• ¿Qué pasa? → Quoi de neuf ?
• ¿Qué hay de nuevo? → Quoi de neuf ?
• ¿Qué haces? → Que fais-tu ?
• Buenos días → Bonjour (le matin)
• Buenas tardes → Bon après-midi / Bonsoir
• Buenas noches → Bonsoir / Bonne nuit
• ¡Ey! / ¡Hey! → Hé !
• ¿Todo bien? → Tout va bien ?
• ¿Cómo andas? → Comment ça va ?

🤝 Salutations formelles
(inconnus, professeurs, personnes âgées, travail)

• Buenos días → Bonjour (le matin)
• Buenas tardes → Bon après-midi / Bonsoir
• Buenas noches → Bonsoir / Bonne nuit
• ¿Cómo está usted? → Comment allez-vous ?
• ¿Cómo se encuentra? → Comment vous sentez-vous ?
• Mucho gusto → Enchanté(e)
• Encantado/a de conocerle → Ravi(e) de vous rencontrer
• Es un placer conocerle → C'est un plaisir de vous rencontrer
• Bienvenido/a → Bienvenue
• ¿Qué tal está usted? → Comment allez-vous ?

👤 Pronoms personnels

Yo → Je
Tú → Tu
Él → Il
Ella → Elle
Usted → Vous (politesse, singulier)
Nosotros / Nosotras → Nous
Vosotros / Vosotras → Vous (pluriel, Espagne)
Ustedes → Vous (pluriel, formel ou Amérique latine)
Ellos → Ils
Ellas → Elles
